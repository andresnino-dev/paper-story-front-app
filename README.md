# Paper Story Front App

Paper Story Front App is a modern React web application for browsing books, viewing book details, managing a shopping cart, and completing a simple checkout flow.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React

## Main Features

- Landing page
- Book catalog with search
- Book detail view
- Shopping cart drawer
- Checkout flow
- Login and profile pages
- Protected routes
